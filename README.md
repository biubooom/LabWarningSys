# LabWarningSys
个人毕设：实验室预警系统

## 项目简介

本系统面向实验室安全监测，结合多种环境传感器实现对温湿度、烟雾/可燃气体、人体活动、环境光照等信息的采集，并可根据阈值或状态变化进行本地显示与预警。

## 传感器与模块信息

| --- | --- | --- | --- | --- | --- | --- |
| DHT22 | 温度、湿度 | `3.3V ~ 6V` | 单总线数字通信 | GPIO | GPIO 输出切换输入，建议开漏输出并外接上拉 | 需要上拉电阻，读取周期不宜过快 |
| HC-SR501 | 人体活动 | `4.5V ~ 20V` | 数字高低电平输出 | GPIO | GPIO 输入，可配浮空输入或上拉输入 | 有预热时间，灵敏度和延时可调 |
| MQ-2 | 烟雾、可燃气体 | `5V` | `AO` 模拟量 / `DO` 数字量 | ADC / GPIO | `AO` 配置为模拟输入，`DO` 配置为输入模式 | 功耗较大，需要预热 |
| DS18B20 | 温度 | `3.0V ~ 5.5V` | 单总线数字通信 | GPIO | GPIO 输出切换输入，建议开漏输出并外接上拉 | 需要上拉电阻，可多点挂载 |
| 光敏电阻模块 | 光照强弱 | `3.3V` 或 `5V` | 模拟量，部分模块带数字量输出 | ADC / GPIO | `AO` 配置为模拟输入，`DO` 配置为输入模式 | 适合判断明暗变化 |

### 1. DHT22 温湿度传感器

- 作用：采集环境温度和湿度
- 供电电压：`3.3V ~ 6V`
- 通信方式：单总线数字通信
- 数据特点：输出数字量，温湿度一体，精度高于 DHT11
- 接线要点：
  - `VCC` 接 `3.3V` 或 `5V`
  - `GND` 接地
  - `DATA` 接 MCU GPIO
  - `DATA` 通常需要外接上拉电阻，常用 `4.7kΩ ~ 10kΩ`
- 引脚配置方式：
  - 数据引脚通常需要在输出和输入之间切换
  - 输出阶段建议使用开漏输出
  - 读取阶段切换为输入模式
- 使用注意：
  - 采样速度较低，不适合高频读取
  - 连续读取间隔通常建议不小于 `2s`

### 2. HC-SR501 人体红外传感器

- 作用：检测人体活动，用于入侵/靠近检测
- 供电电压：`4.5V ~ 20V`
- 输出方式：数字输出
- 通信方式：无通信协议，本质是 GPIO 电平输出
- 数据特点：
  - 检测到人体运动时输出高电平
  - 无人或延时结束后输出低电平
- 接线要点：
  - `VCC` 接电源
  - `GND` 接地
  - `OUT` 接 MCU GPIO 输入口
- 引脚配置方式：
  - `OUT` 对应 MCU 配置为 GPIO 输入
  - 一般可用浮空输入；若线路较长，也可根据实际情况改为上拉输入
- 使用注意：
  - 模块有一定预热时间
  - 板上通常可调灵敏度和延时时间
  - 输出高电平幅值与模块供电有关，接 `3.3V` MCU 时要确认电平兼容

### 3. MQ-2 烟雾/可燃气体传感器

- 作用：检测烟雾、液化气、甲烷、氢气等可燃气体
- 供电电压：`5V`
- 输出方式：
  - 模拟输出 `AO`
  - 数字输出 `DO`（带比较器的模块）
- 通信方式：无通信协议，通常接 ADC 或 GPIO
- 数据特点：
  - `AO` 可用于读取气体浓度变化趋势
  - `DO` 适合做超阈值报警
- 接线要点：
  - `VCC` 接 `5V`
  - `GND` 接地
  - `AO` 接 MCU ADC 输入
  - `DO` 可接 GPIO 输入
- 引脚配置方式：
  - `AO` 配置为模拟输入
  - `DO` 配置为 GPIO 输入，可按需要选择浮空输入或上拉输入
- 使用注意：
  - 加热丝功耗较大
  - 上电后需要预热，初始数据波动明显
  - 更适合做相对变化检测，不适合直接当作高精度浓度计

### 4. DS18B20 温度传感器

- 作用：采集单点温度
- 供电电压：`3.0V ~ 5.5V`
- 通信方式：单总线 `1-Wire`
- 数据特点：输出数字温度值，抗干扰能力较好，可多点挂载
- 接线要点：
  - `VDD` 接 `3.3V` 或 `5V`
  - `GND` 接地
  - `DQ` 接 MCU GPIO
  - `DQ` 需要上拉电阻，常用 `4.7kΩ`
- 引脚配置方式：
  - 数据引脚需要在输出和输入之间切换
  - 输出阶段建议配置为开漏输出
  - 读取阶段切换为输入模式
- 使用注意：
  - 可采用外部供电模式，也可寄生电源模式
  - 多个 DS18B20 可挂在同一总线上，通过 ROM 地址区分

### 5. 光敏电阻模块

- 作用：检测环境光照强弱
- 供电电压：取决于具体模块，常见为 `3.3V` 或 `5V`
- 输出方式：
  - 原始光敏电阻一般输出模拟变化，需要分压后接 ADC
  - 带 LM393 比较器的模块通常同时提供 `AO` 和 `DO`
- 通信方式：无通信协议，通常接 ADC 或 GPIO
- 数据特点：
  - 光线越强，阻值通常越小
  - 更适合判断明暗变化，不适合直接测量标准照度值
- 接线要点：
  - 若是原始光敏电阻，应与固定电阻组成分压电路
  - 若是传感器模块，可将 `AO` 接 ADC、`DO` 接 GPIO
- 引脚配置方式：
  - 原始分压输出或模块 `AO` 均配置为模拟输入
  - 若使用模块 `DO`，则配置为 GPIO 输入
- 使用注意：
  - 环境光变化快时，建议做滤波或多次采样平均
  - 不同模块的灵敏度和阈值范围差异较大

## 本系统可采集的信息

- 温度：`DHT22`、`DS18B20`
- 湿度：`DHT22`
- 人体活动：`HC-SR501`
- 烟雾/可燃气体：`MQ-2`
- 环境光照变化：光敏电阻模块
